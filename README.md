# IconNet

An online icon collecting game — collect, unlock, and compete to own the rarest icons on the network.

---

## About

IconNet is a browser-based collectible game where players unlock icon blocks, open packs, and build out their collection.

## Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React + TypeScript |
| Backend | Node.js + Express |
| Database | MongoDB + Mongoose |
| Auth | JWT + bcrypt |
